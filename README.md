# Girigiri
Shogi AI written in Rust

## Usage

1. [Install Rust](https://www.rust-lang.org/en-US/install.html)
2. Install Girigiri

    ```
    $ git clone https://github.com/bknshn/girigiri_rust
    ```

3. Run
- Run auto battle

    ```
    $ cargo run --bin auto --release
    ```

- Run interpreter

    ```
    $ cargo run --bin debug --release
    ```
